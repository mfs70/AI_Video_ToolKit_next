# AI Video Toolkit (Next)

## 📍 Текущее стабильное состояние

```text
Стабильный плеер, хоткей, драгэнддроп
```

Это контрольная точка проекта.
Любые изменения должны сохранять работоспособность этого состояния.

---

## 🎬 Функциональность плеера

* Загрузка видео:

  * через кнопку Load (Ctrl + L)
  * через Drag & Drop
* Поддерживаемые форматы:

  * `.mp4`, `.mkv`, `.mov`, `.avi`
* При загрузке:

  * отображается первый кадр
  * таймлайн устанавливается в 0
  * плеер находится в состоянии ожидания (Idle)

---

## ▶️ Воспроизведение

* Старт с текущей позиции
* Корректный pause / resume (без сброса позиции)
* Нет скачков курсора
* Нет ускорения после паузы
* После окончания:

  * при выключенном Loop → возврат в начало
  * при включенном Loop → повтор воспроизведения

---

## ⏹ Stop (важное поведение)

Кнопка Stop (S):

* НЕ выгружает файл
* НЕ очищает состояние
* выполняет:

  * сброс позиции в 0
  * обновление кадра предпросмотра
  * переход в Idle

---

## ⏱ Таймлайн

* Отображает текущую позицию
* Позволяет перемещаться мышкой
* При перемещении:

  * в паузе → обновляется кадр
  * в воспроизведении → перезапуск с новой позиции

---

## 🎮 Управление (Hotkeys)

| Клавиша       | Действие            |
| ------------- | ------------------- |
| Space         | Play / Pause        |
| K             | Pause               |
| L             | Ускорение           |
| J             | Сброс скорости (x1) |
| ← / →         | ±1 кадр             |
| Shift + ← / → | ±10 кадров          |
| S             | Stop                |
| R             | Toggle Loop         |
| Ctrl + L      | Load                |

---

## ⚡ Скорость воспроизведения

Шкала скоростей:

```text
x1 → x2 → x4 → x5 → x6 → x7 → x8 → x10 → x16
```

* Индикатор скорости отображается в UI
* При изменении скорости:

  * во время воспроизведения → перезапуск с текущей позиции
* J всегда возвращает x1

---

## 🖱️ Drag & Drop

* Принимается только первый файл
* Игнорируются:

  * папки
  * неподдерживаемые форматы
* После дропа:

  * обновляется таймлайн
  * отображается первый кадр
  * обновляется имя файла
  * состояние → Idle

---

## 🎨 UI поведение

| Состояние | Иконка | Цвет    |
| --------- | ------ | ------- |
| Idle      | ▶      | Белый   |
| Play      | ▶      | Зелёный |
| Pause     | ⏸      | Жёлтый  |

---

## 🧠 Архитектурные принципы

* Воспроизведение управляется через `BufferedVideoPlayer`
* Получение кадров через `FrameGrabber`
* Метаданные через `FFprobeService`

---

## 🔒 Правило стабильности

Если система ломается:

```text
ОТКАТ К: Стабильный плеер, хоткей, драгэнддроп
```

При откате:

* запрещено менять UI
* запрещено менять архитектуру
* возвращается полностью рабочая версия
