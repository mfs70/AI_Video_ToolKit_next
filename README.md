# 🎬 AI Video Toolkit (WPF + FFmpeg)

## 📌 Описание

**AI Video Toolkit** – настольное приложение на базе **WPF (.NET 8)** для работы с видео и изображениями. Использует **FFmpeg** для обработки видео, **FFprobe** для анализа медиафайлов, и архитектуру на основе **MVVM** с внедрением зависимостей (DI).  
В перспективе – добавление AI-модулей (Real-ESRGAN, RIFE).

## 🎯 Текущие возможности

### 📂 Работа с файлами
- Drag & Drop одиночных файлов и папок (включая вложенные)
- Поддержка видео: `.mp4`, `.mkv`, `.mov`, `.avi`, `.webm`
- Поддержка изображений: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.gif`
- Плейлист и монтажный стол

### ▶️ Плеер
- Воспроизведение видео через `BufferedVideoPlayer` (FFmpeg → raw video + PCM audio)
- Аудиовыход через NAudio (стерео PCM 44.1 кГц)
- Управление: ⏮ / ▶ / ⏸ / ⏹ / ⏭
- Регулировка скорости: 0.1x … 16x
- Вкл./выкл. аудио и видео дорожек
- Цикличное воспроизведение

### ✂️ Нарезка и экспорт
- Таймлайн с установкой маркеров (I – вход, O – выход, C – разрез)
- Выделение сегментов, предпросмотр
- Экспорт сегментов в папку `Cut` (кодек H.264, копирование аудио)
- Склейка клипов из монтажного стола (конкатенация)
- Кастомный прогресс-бар с процентом

### 🔍 Анализ медиа
- Получение метаданных через `ffprobe`: разрешение, FPS, кодеки, битрейты, параметры аудио
- Отображение информации в UI

### 📜 Логирование
- Весь вывод FFmpeg/FFprobe перенаправляется в лог-панель

.NET 8, WPF

Microsoft.Extensions.DependencyInjection (DI)

NAudio (аудиовывод)

FFmpeg (внешний процесс)

⚙️ Установка и запуск
Установите .NET 8 SDK (https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

Скачайте FFmpeg (https://ffmpeg.org/download.html)
Рекомендуемый путь: C:\_Portable_\ffmpeg\bin\ffmpeg.exe

Укажите путь в App.xaml.cs (строка ffmpegPath и ffprobePath)

Соберите и запустите:

bash
dotnet build
dotnet run --project AI_Video_ToolKit.UI
